# MedEdge - Landing Page

Prior authorization automation for specialty medical practices.

## Deploy to Netlify

1. Go to [app.netlify.com/drop](https://app.netlify.com/drop)
2. Drag this entire folder onto the page
3. Done. Live URL in seconds.

## Before sharing widely

- Register domain (mededge.ai or mededge.health) and point it at the Netlify site
- Update the email in the CTA (currently hello@mededge.ai)
- Remove robots.txt when ready for search engines to index

## Files

- `index.html` - Landing page (single file, no build step)
- `netlify.toml` - Netlify config with security headers
- `_redirects` - SPA redirect
- `robots.txt` - Blocks crawlers until you're ready to go public
